# Tarea 1: Clasificador Lineal y Regresión con TensorFlow

Este repositorio contiene la primera tarea del curso de **Deep Learning**. En este trabajo se implementaron modelos básicos de aprendizaje supervisado utilizando TensorFlow y NumPy.

## Contenido del Proyecto
1. **Problema 1:** Clasificador lineal con nubes de datos esféricas y línea de decisión personalizada.
2. **Problema 2:** Regresión lineal simple con datos sintéticos y optimización mediante `tf.GradientTape()`.
3. **Problema 3:** Red Neuronal desde cero aplicada al dataset Iris (clasificación binaria).

## Tecnologías Utilizadas
* Python 3
* TensorFlow 2.x
* NumPy
* Matplotlib (para la visualización de resultados)
