# Tarea 1 🤖 Deep Learning: De Puntos a Neuronas 
Bienvenido a mi repositorio de la primera tarea del curso de Deep Learning. En este proyecto, exploro el pipeline completo del aprendizaje automático: desde la generación y preparación de datos hasta la optimización de parámetros y la evaluación de modelos.

## 🎯 Objetivos del Proyecto
El propósito de este trabajo es comprender cómo una máquina "aprende" a decidir y predecir mediante la implementación de tres soluciones fundamentales:

1. **Problema 1:** Clasificador lineal con nubes de datos esféricas y línea de decisión personalizada.
2. **Problema 2:** Regresión lineal simple con datos sintéticos y optimización mediante `tf.GradientTape()`.
3. **Problema 3:** Red Neuronal desde cero aplicada al dataset Iris (clasificación binaria).
